# 주식 거래소 구현 프로젝트

## 프로젝트 개요  
가상의 주식 거래소를 구현한 프로젝트입니다.  
실제 거래소와 유사한 매칭 엔진, 주문 처리 시스템, 실시간 데이터 처리, 계좌 관리 기능 등을 포함하고 있습니다.

- **개발 기간**: 3개월  
- **목표**: 실시간 주식 거래 시스템을 프론트부터 백엔드까지 전체 구현
